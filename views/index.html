Timestamp API

How to use:

Clone the following github repository: https://github.com/akcm25/fcc-api-1

Run npm start on terminal.

Go to localhost:3000


Example:

  • localhost:3000/api/2024-11-13
  • localhost:3000/api/1659008800000

Output:

{"unix":1731456000000,"utc":"Wed, 13 Nov 2024 00:00:00 GMT"}
